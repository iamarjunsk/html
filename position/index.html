This div has Static positioning

Static positioned elements are not affected by the top, bottom, left, and right properties.

This div has Fixed positioning

An element with position:fixed, is positioned relative to the viewport. Which means it always stays in the same place even if the page scrolled.
top, right, bottom, left properties are used to position the element.

This div has Absolute positioning

An element with position:absolute, is positioned relative to the nearest positioned ancestor(insted of positioned relative to the viewport).
However, id an absolute positioned element has no positioned ancestors, it uses the document body, and moved along with page scrolling.

This div has Relative positioning

An element with position:relative, is positioned relative to the normal position
Setting the top, right, bottom, and left properties of a relatively -positioned element will not be adjusted to fit into any gap left by the elemtnt.
